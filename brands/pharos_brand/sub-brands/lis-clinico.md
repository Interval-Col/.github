## 2. Positioning

The clinical laboratory information system for **Laboratorio Clínico Hematológico** — result
entry/validation, pathology, surveys, consents, sample-taking. **QC is a surface *inside* this
LIS** (RFC 0004 §2), and its **burgundy/red survives as the QC state palette** (en-control /
fuera-de-control / alerta) — a recognizable status language, **not** the app accent.

## 3. Visual (overrides only)

- **Accent** `--primary` = teal profundo `#1B6B5A` (light) / `#4CD1B0` (dark). One colour from
  the curated family set; status colours are **accent-independent** (RFC 0008 Q4).
- **Glyph** the «Sonda» radar mark (lucide `Radar`) as the app icon; maritime identity lives in
  the mark, so the name stays the plain "Clínico".
- **Lockup** `Pháros` + `·` + Sonda glyph + `Clínico` + optional sublabel
  `LIS · LABORATORIO CLÍNICO` (IBM Plex Mono, uppercase, tracked, muted).
- Everything else (type, shell, spacing, flatness) **inherited unchanged** from `../BRAND.md`.

## 4. Voice

Inherits the shared Pháros voice (Spanish-first, technically precise, warm, never alarmist).
Domain register: clinical/lab — precise about result states, never alarmist about patient data.
